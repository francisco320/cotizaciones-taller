# Guía de diseño – Cotizaciones

## Paleta
- **Primario:** `#0f766e` (teal-700) – botones principales, enlaces, títulos.
- **Primario hover:** `#0d9488` (teal-600).
- **Fondo:** `#f8fafc` (slate-50) para páginas; cards en blanco con sombra suave.
- **Texto:** `#0f172a` (slate-900) títulos, `#475569` (slate-600) cuerpo.
- **Bordes:** `#e2e8f0` (slate-200).

## Tipografía
- **Títulos:** font-semibold o font-bold, text-slate-900.
- **Cuerpo:** text-slate-600, text-sm o text-base.
- **Números/totales:** tabular-nums, font-medium.

## Componentes
- **Cards:** bg-white, rounded-lg, shadow-sm, border border-slate-200, padding estándar.
- **Inputs:** border rounded-md, focus:ring-2 focus:ring-teal-500/20 focus:border-teal-500.
- **Botones primarios:** bg-teal-600 hover:bg-teal-700 text-white.
- **Botones secundarios:** border border-slate-300 bg-white hover:bg-slate-50.

## Layout
- Contenedor principal: max-w-6xl mx-auto px-4.
- Formularios en secciones con títulos (Empresa, Cliente, Vehículo, Ítems, Totales).
- Tabla de ítems: cabecera gris clara, filas alternas opcionales.
